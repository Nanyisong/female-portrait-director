# FAQ

## 为什么要参数锁定？

参数锁定用于确保用户明确填写的风格、场景、服装、气质、镜头、光线和用途不会被系统擅自替换。系统只做细化、补全和稳定化。

## 为什么必须有五官模块？

五官模块可以降低人物长相趋同、网红脸、假面感和过度磨皮的问题，让不同提示词之间保留差异。

## 为什么纯欲风不能写得太露骨？

该风格的目标是明确成年女性的时尚写真和克制吸引力。露骨、色情、挑逗或年龄模糊表达会增加安全风险，也容易让生成结果失控。

## 为什么电商图要优先服装？

电商图片的核心目标是展示商品。服装颜色、版型、材质、纹理和细节必须优先于人物表现和背景氛围。

## 为什么不要把所有风格词堆在一起？

冲突的风格词会让场景、光线、服装和构图失去明确目标，导致画面廉价、杂乱或生成不稳定。每次只选择一个主路由。
