# 版本管理

当前公开版本：

```text
FEMALE-PORTRAIT-DIRECTOR-V2.0-MODULAR
```

## V2.0-MODULAR 变更摘要

- 使用 `skill/style-registry.md` 作为唯一风格分流入口。
- 将底层规则拆分到 `skill/core/`，统一参数锁定、安全边界、输出格式、冲突处理和缺失参数补全。
- 将 Route 按 lifestyle、fashion、oriental、fantasy、commercial 和 curve 分类。
- 将已实现 Route 从 5 条扩展到 14 条，并为后续扩展保留注册占位。
- 保持导演式扩写：先完成视觉推导，再生成完整提示词，不做摘要式或机械填参式输出。

## V1.3 变更摘要

- 恢复默认 `24–28 岁年轻成年东方女性` 的可见年龄特征。
- 明确标准输出不是摘要，必须逐项扩写年龄、五官、身形、姿态、服装、场景、镜头、光线和滤镜。
- 将视觉补全库调整为标准输出必读文件。
- 为滤镜增加色温、饱和度、对比度、高光、暗部、肤质和颗粒等可执行参数。
- 保留导演式瞬间设计，但将其定位为模块融合层。

## V1.2 变更摘要

- 将单体核心规则拆分为轻量调度器、五类按需路由、导演式扩写规范和视觉补全库。
- 新增生活瞬间、动作链、视线落点、选择性环境细节和平台用途转换规则。
- 统一安全边界为单一权威文件。
- 统一参数别名和示例命名，减少规则漂移。
- 新增 `简洁`、`标准`、`导演详版` 和 `直接生成图片` 输出模式。

## 版本格式

```text
FEMALE-PORTRAIT-DIRECTOR-V主版本.次版本
```

- 主版本：核心结构、路由规则或输出契约发生不兼容变化时递增。
- 次版本：新增风格、补充示例、优化措辞或改进安全表达时递增。
