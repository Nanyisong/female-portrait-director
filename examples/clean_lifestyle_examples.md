# 清纯生活照示例

## 参数组合

```text
写真风格：清纯生活照
场景方向：咖啡馆靠窗座位
服装方向：白色针织开衫 + 浅色内搭
气质标签：清纯温柔、自然亲近
五官方向：初恋感淡颜
身形方向：自然协调身形
镜头方向：自然坐姿，大腿以上构图
光线氛围：柔和自然光
滤镜效果：奶油暖白生活剧照滤镜
画幅比例：9:16
平台用途：生活照 / 小红书种草
补充要求：真实自然，不要过度摆拍
```

## 最终提示词

```text
生成一张 9:16 竖版、真实摄影质感的女性生活照，整体为温柔自然的清纯生活照风格。画面主体是一位明确成年的东方女性，具有自然清秀的东方女性特征，成年气质明确，不幼态化，没有未成年感。人物气质清纯温柔、自然亲近，像真实生活电影中被捕捉到的一帧日常片段。

人物五官方向为初恋感淡颜，脸型柔和流畅，五官清秀自然，眼神干净温柔，鼻型秀气，唇形自然，妆容清透淡雅，不网红脸，不假面感。人物身形自然协调，姿态简单放松。

场景位于真实咖啡馆的靠窗座位，环境干净、有生活气息，可见浅色桌面、咖啡杯和柔和室内背景，但不杂乱。人物穿着白色针织开衫和浅色内搭，服装完整、柔软、干净。人物自然坐着，采用大腿以上构图。光线为柔和自然光，整体使用奶油暖白生活剧照滤镜，画面温柔、通透、真实。
```
