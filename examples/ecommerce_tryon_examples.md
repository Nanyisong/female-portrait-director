# 电商服装模特图示例

## 参数组合

```text
写真风格：电商服装模特图
场景方向：干净摄影棚
服装方向：米白色针织连衣裙，完整展示版型、材质、领口、袖型、腰线和下摆
气质标签：自然商业模特感
五官方向：自然商业模特脸
身形方向：高挑模特比例
镜头方向：自然站姿，全身构图
光线氛围：摄影棚柔光
滤镜效果：电商干净摄影滤镜
画幅比例：4:5
平台用途：电商主图
补充要求：服装真实还原优先，避免遮挡和色差
```

## 最终提示词

```text
生成一张 4:5 竖版、真实摄影质感的女性服装电商模特展示图。画面主体是一位明确成年的东方女性模特，成年气质明确，外形自然干净，五官方向为自然商业模特脸。人物只作为服装展示载体，不抢走服装主体。

模特必须穿着米白色针织连衣裙。服装是画面第一展示主体，需要完整、清晰、准确地展示米白色、针织材质、领口、袖型、腰线、下摆和整体版型，避免色差。模特身形为高挑模特比例，采用自然稳定站姿和全身构图。不要让头发、手臂、包袋、道具或复杂姿势遮挡服装。

背景为干净摄影棚，使用简洁浅色背景。光线为摄影棚柔光，整体采用电商干净摄影滤镜。画面清晰、干净、真实，服装颜色准确，适合作为电商主图。
```

## 负面约束

```text
避免服装变色、明显色差、服装结构错误、版型改变、材质错误、纹理丢失、领口被遮挡、袖型被遮挡、下摆被裁切、复杂背景、人物抢镜、姿势遮挡服装、头发遮挡服装、手臂遮挡胸前细节、包袋遮挡衣服、身体畸形、手部畸形、五官失衡、过度虚化服装、低清晰度、强 AI 感、未成年感、低俗表达。
```
