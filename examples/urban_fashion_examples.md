# 都市时尚写真示例

## 参数组合

```text
写真风格：都市时尚写真
场景方向：城市写字楼落地窗旁
服装方向：黑色西装外套 + 白色丝质内搭 + 高腰长裤
气质标签：清冷、利落、成熟自信
五官方向：清冷高级脸
身形方向：修长匀称
镜头方向：自然站姿，大腿以上构图
光线氛围：冷白自然光
滤镜效果：都市高级灰摄影滤镜
画幅比例：4:5
平台用途：社交媒体 / 公众号配图
补充要求：现代、简洁，不要廉价影楼感
```

## 最终提示词

```text
生成一张 4:5 竖版、真实摄影质感的都市女性时尚写真。画面主体是一位明确成年的东方女性，成年气质明确，整体清冷、利落、成熟自信。人物五官方向为清冷高级脸，面部轮廓自然清晰，眼神克制坚定，妆容干净精致，不网红脸，不假面感。

人物身形修长匀称，采用自然站姿和大腿以上构图。人物穿着黑色西装外套、白色丝质内搭和高腰长裤，服装结构完整，线条简洁。场景位于城市写字楼落地窗旁，现代建筑背景轻微虚化，画面秩序清晰，不使用杂乱街景。

光线为冷白自然光，使用都市高级灰摄影滤镜，保留真实皮肤质感和服装材质细节。整体呈现现代、成熟、有距离感的都市时尚摄影氛围。
```

## 负面约束

```text
避免未成年感、幼态脸、过度少女化、网红整容脸、假面感、过度磨皮、廉价影楼风、杂乱背景、夸张姿态、服装结构错误、肢体畸形、手部畸形、光线过曝、画面脏灰、低清晰度、强 AI 感、CG 插画感。
```
