# 参数字段

系统必须锁定用户明确填写的字段，只对缺失字段执行合理补全。

| 字段 | 说明 | 推荐选项 | 默认值 |
| --- | --- | --- | --- |
| 写真风格 | 选择主风格路由 | 清纯生活照、纯欲曲线生活照、都市时尚写真、古风仙侠美人图、电商服装模特图 | 根据用户意图判断；无法判断时使用清纯生活照 |
| 场景方向 | 指定拍摄环境 | 咖啡馆、海边步道、城市街头、写字楼、摄影棚、古风庭院、山水云雾场景 | 与主风格协调的简洁场景 |
| 服装方向 | 描述颜色、版型、材质和搭配 | 生活穿搭、时尚通勤、古风服饰、电商待展示服装 | 与主风格匹配的完整服装 |
| 气质标签 | 描述人物整体气质 | 清纯温柔、安静克制、清冷利落、成熟自信、疏离仙气、自然商业模特感 | 自然、干净、明确成年 |
| 五官方向 | 控制脸型与面部特征 | 邻家清秀脸、初恋感淡颜、轻熟知性脸、清冷高级脸、自然商业模特脸 | 自然东方女性五官 |
| 身形方向 | 控制整体比例 | 自然协调身形、修长匀称、健康紧致、高挑模特比例、自然吸引力曲线 | 自然协调身形 |
| 身形吸引力强度 | 控制线条表达强度 | 低、中、高；始终保持成年、时尚、克制 | 低或中 |
| 线条重点 | 指定适度强调区域 | 肩颈、锁骨、腰线、腿部比例、整体轮廓 | 与服装和镜头匹配 |
| 镜头方向 | 控制景别和构图 | 面部特写、半身、大腿以上、全身、轻侧身、自然坐姿 | 半身或大腿以上 |
| 光线氛围 | 控制光源与情绪 | 柔和自然光、冷白自然光、傍晚柔光、摄影棚柔光、古风月白柔光 | 柔和自然光 |
| 滤镜效果 | 控制画面质感 | 奶油暖白、冷白生活照、都市高级灰、古风清冷、商业干净摄影 | 与主风格匹配 |
| 画幅比例 | 控制成图比例 | 9:16、4:5、3:4、1:1、16:9 | 9:16 |
| 平台用途 | 控制构图目标 | 生活照、小红书种草、社交媒体、公众号配图、电商主图、电商详情页 | 根据场景判断 |
| 补充要求 | 用户的额外限制 | 构图、背景、姿态、颜色、展示重点等 | 保持真实、克制、完整服装 |
