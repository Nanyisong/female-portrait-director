# 公开调用说明

## Skill 是什么

女性人像提示词导演 Skill 用于将少量创作要求整理为稳定、完整、可直接复制的女性人像 AI 生图提示词。系统支持自动补全五官、身形、镜头、光线、滤镜、平台用途和负面约束。

## 如何输入参数

最简调用只需提供以下字段：

```text
风格：
场景：
服装：
气质：
画幅：
```

需要更精细控制时，可以使用 [parameter_schema.md](parameter_schema.md) 中的完整字段。

## 支持风格

1. 清纯生活照
2. 纯欲曲线生活照
3. 都市时尚写真
4. 古风仙侠美人图
5. 电商服装模特图

## 示例调用

```text
风格：都市时尚写真
场景：城市写字楼落地窗旁
服装：黑色西装外套 + 白色丝质内搭 + 高腰长裤
气质：清冷、利落、成熟
镜头：大腿以上构图
光线：冷白自然光
画幅：4:5
```

## 输出格式

```text
一、参数锁定结果
二、风格判断
三、最终提示词
四、负面约束
五、可选优化建议
```

## 使用边界

- 默认人物必须是虚构、明确成年的女性。
- 仅生成非裸露、非色情、非伤害性的视觉提示词。
- 不得用于未成年人性化、年龄模糊人物性化、非自愿图像、欺骗性身份内容或侵犯隐私的用途。
- 使用者需要遵守目标 AI 平台政策和所在地区法律法规。
